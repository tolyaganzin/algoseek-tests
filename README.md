# how to check it

- clone repo
- open terminal at project folder
- run `cd algoseek-tests`
- run `npm i --legacy-peer-deps`
- run `ng serve`
- navigate to `http://localhost:4200/` at your browser
- select any of the 3 tests for checking result

- main menu
- test 1
- test 2
- test 3
